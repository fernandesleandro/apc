In this folder you can add local gallery images for projects.

Recommended structure:

- `public/images/gallery/residencial-niemeyer/1.jpg`
- `public/images/gallery/residencial-niemeyer/2.jpg`
- `public/images/gallery/cosmopolitan/1.jpg`

Naming suggestions:
- Use sequential numbers for ordering (1.jpg, 2.jpg, ...)
- Keep optimized images (max width 1600px) and use `webp` when possible

After adding images, ensure `data/database.json` references the correct paths in the `galleries` collection.

You can remove this README after uploading your images.
